# Daily Textbook Chapters

These chapters expand the five-day course outline into day-specific student
readings. They are meant to be assigned one day at a time.

- [Day 1: APIs and Platform Data Access](day_1_apis.md)
- [Day 2: Static Web Scraping Foundations](day_2_static_scraping.md)
- [Day 3: Dynamic Pages and Fragile Scrapers](day_3_dynamic_scraping.md)
- [Day 4: DSA Access and Platform Research Tools](day_4_dsa_platform_tools.md)
- [Day 5: AI-Augmented and Reproducible Workflows](day_5_ai_reproducibility.md)

Each chapter follows the same logic: conceptual framing, technical foundations,
methodological risks, guided practice, and end-of-day reflection questions.
